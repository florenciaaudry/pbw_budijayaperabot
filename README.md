# PBW – Perabot Budi Jaya

Website profil dan katalog sederhana untuk Perabot Budi Jaya Marelan.
Project ini dibuat sebagai tugas mata kuliah Pemrograman Berbasis Web (PBW).

Website dibangun menggunakan PHP native dengan konsep modular (include),
serta HTML, CSS, dan JavaScript.

## Teknologi
- PHP
- HTML
- CSS
- JavaScript

## Struktur Project
- header.php  
- navbar.php  
- index.php  
- footer.php  
- style.css  
- script.js  
- folder gambar

## Cara Menjalankan
1. Install XAMPP
2. Pindahkan folder project ke dalam folder `htdocs`
3. Jalankan Apache
4. Buka browser dan akses `http://localhost/nama-folder/`
